  マークダウン翻訳では、コードやリンクは翻訳されず、翻訳された文がキャッシュされます。翻訳が変更されても元のテキストが変更されていない場合、再実行しても翻訳の変更は上書きされません。

  `yaml`で生成された Web サイトを翻訳するための言語ファイル。
